Welcome to CalcTiga

CalcTiga adalah kalkulator berbasis Web Apps untuk membantu User menghitung Luas Segitiga & Keliling Segitiga gratis. Silahkan masukkan angka anda di kotak bawah untuk mencobanya.

Luas Segitiga

Rumus Luas Segitiga:

L = ½ x a x T


Keterangan Rumus:

L = Luas segitiga

a = Alas segitiga

T = Tinggi segitiga

Keliling Segitiga

Rumus Keliling Segitiga:

K = A + B + C


Keterangan Rumus:

A = Sisi Segitiga ke-1

B = Sisi Segitiga ke-2

C = Sisi Segitiga ke-3
